# Контракты -> Excel

Десктопное приложение для извлечения данных из электронных контрактов (HTML-файлы с сайта zakupki.gov.ru / ЕИС) и переноса в Excel.

## Возможности

- Парсинг HTML-файлов печатных форм контрактов с zakupki.gov.ru
- Поддержка PDF-файлов (с текстовым слоем)
- Перетаскивание файлов (drag-and-drop) или выбор через проводник
- Запись данных в существующий Excel файл или создание нового
- Удобный графический интерфейс

## Извлекаемые данные

| Поле | Описание |
|------|----------|
| Собственник товара | Аббревиатура заказчика (МЗ) |
| Тип | Тип заявки |
| № договора | Номер контракта и дата |
| № аукциона | Номер извещения о закупке |
| МНН | Международное непатентованное название |
| Торговое наименование | Торговое название препарата |
| Лек. Форма | Лекарственная форма и дозировка |
| Ед. изм. | Единица измерения (упак.) |
| Общее кол-во | Количество упаковок |
| Производитель | Держатель РУ / Производитель |
| Цена ед. продукции | Цена за упаковку |
| Общая цена | Итоговая сумма контракта |
| Поставщик | Наименование поставщика |
| Способ размещения | Способ закупки |

## Установка и запуск

### Из исходного кода (Python 3.10+)

```bash
pip install -r requirements.txt
python main.py
```

### Готовый .exe (Windows)

Скачайте `ContractParser.exe` из раздела [Releases](../../releases) и запустите.

## Сборка .exe

### Локально (Windows)

```bash
pip install -r requirements.txt
pyinstaller --onefile --windowed --name ContractParser main.py
```

Готовый файл будет в папке `dist/ContractParser.exe`.

### Через GitHub Actions

При создании тега (например `v1.0.0`) автоматически запускается сборка .exe и создаётся Release.

```bash
git tag v1.0.0
git push origin v1.0.0
```
